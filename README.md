# 地球 Online：补丁说明

一款补丁运营轻策略游戏。玩家不是神，而是地球 Online 的值班运营：每个版本从候选补丁里挑一个上线，撑过 12 个版本，同时完成季度目标、压住事故热度、管理预算和玩家口碑。

## 当前版本

- 核心循环：选补丁、完成季度验收、处理事故、用紧急操作救场。
- 资源系统：口碑、预算、维护窗、回滚令、事故热度。
- 连携系统：连续做同类补丁会触发专题二连，但连续太久会让玩家疲劳。
- 事故系统：高风险补丁会推高事故热度，热度过高会触发随机事故。
- 紧急操作：开维护窗、紧急热修、回滚上一版。

## 打开方式

用 Godot 4 打开本目录，运行 `scenes/main.tscn`。
